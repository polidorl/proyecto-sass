 Este proyecto presenta el diseño de una página web estándar
 como plataforma para demostrar técnicas de programación Sass
 en escenarios reales.Sass es un preprocesador para CSS que
 amplia su funcionalidad con características como el uso de
 variables,anidación, parciales, importaciones, mixins,
 funciones.
 Lo que permite escribir el código CSS más organizado,
 mantenible y reutilizable,fomentando así
 una experiencia de desarrollo más eficiente y escalable.
